# All Care Central — Sprint 1

Lightweight static frontend for the All Care Therapies clinic hub. Built with HTML, CSS, and vanilla JavaScript.

Structure
- index.html — Landing dashboard (Home)
- quick-launch.html — Quick actions
- resources.html — Clinic resources
- css/design-system.css — Shared design system
- js/main.js — Shared JS: UI helpers
- js/ui.js — Small UI helpers
- assets/logo.svg — Branding asset

How to run
- Open `index.html` in a browser. No server required.

Notes
- Authentication removed for development — no login required.
- `index.html` is the landing dashboard with centered All Care Therapies of Georgetown logo.
- Quick Launch uses clean SVG icons (stylized placeholders) for each product.
- "Recently Opened Resources" are tracked in `localStorage` and appear on the dashboard when resources are opened.
- Designed to be responsive and Apple-inspired with Raleway typography.
